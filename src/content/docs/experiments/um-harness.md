# UM_HARNESS

under construction

## Status

Active validation mechanism.

## Research question

Can a generated user manual become a testable contract rather than a passive document?

## Generated system

UM_HARNESS converts user-facing workflows into executable sandbox checks.

## Validation

The mechanism validates commands, inputs, outputs and documented workflows after the user manual is generated.

## Findings

Documentation becomes more valuable when it can be executed and checked against the generated software.

## Metrics

- Artifact tested: user manual
- Validation mode: sandbox workflow
- Role: post-generation quality gate
